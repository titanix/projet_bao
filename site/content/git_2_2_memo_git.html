Création d’un repository pour le dossier courant (récursif)
git init
Ajout d’un fichier aux fichiers à ajouter au dépot au prochain commit
git add <filename>
Voir quel sont les fichiers qui seront ajoutés au prochain commit
git status
Sauvegarde d’un état (commit)
git commit –m "&l;message>"
Revenir à la dernière version sauvegardée d’un fichier
git checkout -- <filename>
Corriger le message du derneir commit
git commit --amend -m "Fichier de sortie encodé en UFT8."
Lister la configuration git config -l Réécrire l’email de la configuration git config --global user.email Ajout du répertoire distant sous le raccourci githubrep git remote add githubrep https://github.com/titanix/projet_bao.git Liste les répertoires distants enregistrés (alias seulement) git remote Listes les répertoires distants enregistrés (noms courts et adresse complètes) git remote -v
